## Getting Started

### Prerequisites

```bash
python 3
```

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/beterrabaA/aladin_api_test
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt 
   ```
3. Run the application 
    ```bash
    python manage.py runserver
   ```
